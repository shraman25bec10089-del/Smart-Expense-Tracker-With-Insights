## 📌 Overview

Smart Expense Tracker is a Python-based command-line application that helps users track daily expenses, analyze spending habits, and generate professional reports.

It combines data storage, visualization, and reporting into one simple tool — making personal finance management easy and effective.

---

## ✨ Features

- 💸 Add and manage daily expenses  
- 📂 JSON-based data storage  
- 📊 Monthly trend visualization  
- 🥧 Category-wise spending analysis  
- 📄 Automatic PDF report generation  
- 🧾 Complete expense history  
- ⚡ Fast and simple CLI interface
